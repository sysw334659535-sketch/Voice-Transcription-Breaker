# Voice-Transcription-Breaker-
# 🎙️ 同期声智能断句工具 (Voice Transcription Breaker)

**同期声智能断句工具** 是一款基于 DeepSeek 大语言模型开发的本地化桌面 GUI 应用，专为新闻媒体工作者、记者和视频后期剪辑师打造。适用于不联网的非线性编辑系统打唱词省点时间，内置deepseek-coder-1.5b-base，启动会释放到缓存目录等待调用，可能会有点慢。

在新闻制作和视频剪辑中，长段的采访录音（同期声）往往需要被切分成适合做字幕的短句。传统工具大多只能“按字数”死板截断，导致主谓宾分离或语义破碎。本工具通过引入本地部署的 LLM，实现了**真正的“自然语义级”断句**。

---

## ✨ 核心特性

* 🧠 **语义级智能断句**：彻底告别机械截断。模型能够精准识别主谓宾结构、并列短语和修饰成分，在保证单行字数（默认 15-18 字）的同时，确保断句符合人类自然朗读习惯。
* 🎯 **智能标签识别**：专为新闻稿件优化，自动识别稿件中的 `【同期声】` 或 `【同期】` 标签，仅对目标内容进行提取和断句，不破坏导语、配音等原稿结构。
* 🛡️ **极致隐私与安全**：**完全离线运行**。程序在代码底层强行禁用了 Socket 网络通信，并支持 CPU 动态量化推理（qint8），确保未公开的新闻稿件和机密素材绝对不会泄露。
* ⚡ **平滑的处理架构**：内置多线程异步队列调度。即使面对万字长文，也能保持 UI 界面流畅响应，避免多任务并发导致的 CPU 死锁和极度卡顿。
* 🛠️ **深度文本清洗**：内置强大的正则后处理引擎。自动剔除无用标点（支持双斜杠 `//` 转换换行），智能合并语气词（如“啊”、“呢”自动与下文合并），并防止介词（如“在”、“从”）错误出现在行首。

## 🛠️ 环境依赖与前置准备

### 1. 基础环境
* **Python**: 3.8 或更高版本
* **操作系统**: Windows (推荐), macOS, Linux

### 2. 下载离线模型
本工具默认使用 [DeepSeek](https://huggingface.co/deepseek-ai) 系列开源模型。为了保证工具的离线运行，请提前将模型下载到本地：
1. 前往 Hugging Face 或 魔搭社区 (ModelScope)。
2. 下载所需模型（如 `deepseek-coder-1.5b-base` 或其他适用的小参数模型）。
3. 将模型文件夹存放在本地，例如：`D:/models/deepseek-coder-1.5b-base`。
