# ⚡ 同期声智能断句工具 | Voice Transcription Breaker

> **“让离网的新闻稿件的断句不再费时费力（无需网络）。”**

---

## 🛠️ 功能概览

| 模块 | 功能特性 | 价值 |
| :--- | :--- | :--- |
| **智能识别** | 双轨制解析（冒号/空格） | 告别繁琐的手动断句 |
| **语法保护** | 词性/气口逻辑保护 | 确保配音语感连贯 |
| **灵活模式** | 同期声/全文本模式切换 | 覆盖多场景办公需求 |
| **极简 UI** | 现代扁平化界面 | 零配置，开箱即用 |

---

## 🚀 极速上手

1. **启动：** 双击 `同期声智能断句工具.exe`。
2. **输入：** 将通稿粘贴至左侧输入框。
3. **设置：** - ✅ **勾选**：仅处理同期声段落。
   - ⬜ **取消**：全文本一视同仁断句。
4. **处理：** 点击 **[⚡ 极速智能断句]**，结果瞬间呈现。
5. **交付：** 点击 **[一键复制结果]**，粘贴至剪辑字幕轨。

---

## ⚠️ 重要说明

* **关于依赖：** 为保证程序极致精简，打包版本已剔除冗余库。若运行提示缺少 `jieba` 分词库，请自行在环境安装，或参考根目录提供的 `requirements.txt` 文件进行修复。

---

## 💡 为什么选择它？

* **针对广电环境深度定制**：原生支持同期声的各种非规范写法（如头衔过长、无冒号等）。
* **微观保护机制**：采用 `jieba` 分词引擎，自动识别句子逻辑，**绝不出现“和”、“的”、“在”等词被切断在行尾的尴尬**。
* **低功耗运行**：纯 Python 标准库 + 极简打包，占用内存极低。
