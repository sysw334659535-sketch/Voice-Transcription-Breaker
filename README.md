同期声智能断句工具 (Voice Transcription Breaker)
简介
本项目是一款专为广电新闻工作者和后期剪辑师打造的离线自动化断句工具。它能够智能解析新闻稿件中的“同期声”片段，并根据 18 字的阅读舒适度进行自动断句与排版，但仍存在一些小Bug。

核心功能
双轨制智能识别： 完美支持“姓名+冒号”或“姓名+空格”两种常见的同期声书写格式。

微观词性保护： 内置语法保护引擎，确保断句不会切断连词（如“因为”、“为了”）或动词短语，确保后期配音读起来顺口。

全场景适配： 支持“仅处理同期声”模式与“全文本通用断句”模式，满足多种办公需求。

极简交互： 一键断句、一键清空、一键复制，零配置即可使用。

快速上手
下载： 获取并运行最新版的 .exe 可执行文件。（环境包精简版可能提示缺少jieba分词库，自己解决吧，提供了备用requirements文件）

输入： 将新闻通稿粘贴至左侧输入框。

设置： 若仅需处理同期声，请勾选“仅处理【同期/同期声】段落”；若需处理全文，请取消勾选。

断句： 点击“⚡ 极速智能断句”按钮。

复制： 点击“一键复制结果”即可获取整理好的文稿，直接用于剪辑软件的字幕导入。

技术栈
GUI 框架： Python Tkinter

分词引擎： Jieba (中文语义断句保护)

打包工具： PyInstaller

开发指南 (开发者专用)
如果你需要重新打包该项目：

安装依赖：pip install -r requirements.txt

执行打包命令：

Bash


pyinstaller -F -w -i "logo.ico" --clean main.py
打包完成后，执行文件将在 dist 文件夹内生成。
